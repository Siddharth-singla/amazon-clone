# 🛒 Amazon Clone (Frontend)

A simple **Amazon homepage clone** built using **HTML and CSS**.
This project replicates the UI of Amazon’s landing page including navbar, hero section, product categories, and footer.

---

## 🚀 Features

* 🧭 Navigation bar with logo, search bar, and cart
* 📍 Location section (India)
* 🔍 Search input UI
* 🖼️ Hero banner section
* 🛍️ Product category cards (Clothes, Electronics, etc.)
* 📦 Grid layout for shopping section
* 📱 Clean and structured UI
* 🦶 Footer with multiple sections

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Font Awesome (for icons)
